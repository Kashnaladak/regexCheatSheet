Regex Cheat Sheet

Pattern Matches Rule
X X Any character except [\^$.|?*+()
XY XY Any character except [\^$.|?*+()
\+ + \ followed by a special character: [\^$.|?*+(){}
\n \n (line feed) Any character except [\^$.|?*+()
\xE7 ç Any hexadecimal represented character
\u20AC € Any unicode represented character
int [ij] int i or int j Any single character within the character class [..]
int [^b] int a, int c, int d, .. Any single character not in the character class [^..]
int i = [0-9] int i = 0, .., int i = 9 Any character in the character class range [..-..]
int [a-c[x-z]] int a, .., int c, int x, .., int z Any character in the union of the classes [..[..]]
int [a-e&&[^c]] int a, int b, int d, int e Any character in the intersection of the classes [..&&[..]]
\d 0, 1, .., 9 Any character that belongs to the digit character class
\p{Digit} 0, 1, .., 9 Any character that belongs to the digit character class
\p{InGreek} Ͱ,ͱ,Ͳ,ͳ,..,Ͼ,Ͽ Any character in the "Greek" unicode page.
. ... Any character except \n
A|B A or B Either expression to the side of the |, but not both
^ Start of a line
$ End of a line
\b Either the beginning or end of a word
\A Beginning of an input
\z End of an input
a* aaaaa... Any character, zero or more times
a+ aaaaa... Any character, one or more times
a? a Any character, zero or one time
a{1,3} a, aa, aaa Any character, at least 'n' times and at most 'm' times
a{2,} aa, aaa, aaaa, .. Any character, at least 'n' times
a{5} aaaaa Any character, exactly 'n' times
<some quantifier>? Reluctant modifier, tries to match as little as possible
<some quantifier>+ Possessive modifier, never tracks back
\QhelloWorld();\E helloWorld(); Pattern quoting
(a)b ab Capturing group on a section of the pattern
(?:a)b ab Non-capturing group on a section of the pattern
(?idm-sux) Activating/deactivating flags
(?=X)Y Y Y, so long as X is true in Y
(?!X)Y Y Y, so long as X is false in Y
(?<=X)Y Y Y, so long as X occurs before Y
(?<!X)Y Y, so long as X doesn't occur before Y
